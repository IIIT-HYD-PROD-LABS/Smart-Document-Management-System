**1. Update frontend/src/lib/api.ts:**

Replace the existing simple 401 response interceptor with the queue-based refresh pattern from the research. The complete interceptor logic:

- Add state variables at module level (above the interceptor):
  ```typescript
  let isRefreshing = false;
  let failedQueue: Array<{
      resolve: (token: string) => void;
      reject: (error: any) => void;
  }> = [];

  const processQueue = (error: any, token: string | null = null) => {
      failedQueue.forEach((prom) => {
          if (error) {
              prom.reject(error);
          } else {
              prom.resolve(token!);
          }
      });
      failedQueue = [];
  };
  ```

- Replace the response interceptor (`api.interceptors.response.use(...)`) with:
  - On success: pass through.
  - On error with 401 status AND `!originalRequest._retry`:
    - If `isRefreshing` is true: queue the request (return a new Promise that pushes to `failedQueue`, and when resolved, retries `originalRequest` with new token).
    - Otherwise: set `originalRequest._retry = true`, `isRefreshing = true`.
    - Try: get `refresh_token` from `Cookies.get("refresh_token")`. If missing, throw.
    - POST to `${API_URL}/api/auth/refresh` using a raw `axios.post` (NOT the `api` instance -- to avoid infinite interceptor loop). Send `{ refresh_token: refreshToken }`.
    - On success: update cookies (`Cookies.set("token", data.access_token, { expires: 1, sameSite: "Strict" })`, `Cookies.set("refresh_token", data.refresh_token, { expires: 7, sameSite: "Strict" })`).
    - Update `api.defaults.headers.common.Authorization`.
    - Call `processQueue(null, data.access_token)`.
    - Retry the original request with new token.
    - On catch: call `processQueue(refreshError, null)`. Remove all cookies (token, refresh_token, user). Redirect to `/login`.
    - Finally: set `isRefreshing = false`.

- Add authApi methods for the new endpoints:
  ```typescript
  refresh: (refreshToken: string) =>
      axios.post(`${API_URL}/api/auth/refresh`, { refresh_token: refreshToken }),
  logout: (refreshToken: string) =>
      api.post("/auth/logout", { refresh_token: refreshToken }),
  ```
