**1. Create backend/app/utils/rate_limiter.py (NEW FILE):**

```python
from slowapi import Limiter
from slowapi.util import get_remote_address
from app.config import settings

limiter = Limiter(
    key_func=get_remote_address,
    storage_uri=settings.REDIS_URL,
    default_limits=[settings.RATE_LIMIT_DEFAULT],
    headers_enabled=True,
)
```

This creates a module-level `limiter` instance that can be imported by any router.

**2. Create backend/app/utils/logging.py (NEW FILE):**

Implement the `setup_logging()` function exactly as in the research Pattern 5. Key points:
- Shared processors: `merge_contextvars`, `add_logger_name`, `add_log_level`, `PositionalArgumentsFormatter`, `ExtraAdder`, `drop_color_message_key` (custom processor to remove uvicorn's color_message), `TimeStamper(fmt="iso", utc=True)`, `StackInfoRenderer`.
- If `settings.LOG_JSON_FORMAT`: use `JSONRenderer` and add `format_exc_info` processor.
- If not: use `ConsoleRenderer` (pretty colored output for development).
- Configure structlog with `ProcessorFormatter.wrap_for_formatter`.
- Set up stdlib logging: clear root handlers, add StreamHandler to stderr with structlog formatter.
- Quiet noisy loggers: `uvicorn.access` to WARNING, `sqlalchemy.engine` to WARNING.

**3. Create backend/app/middleware/__init__.py (NEW FILE):**

Empty `__init__.py` to make `middleware` a Python package.

**4. Create backend/app/middleware/security_headers.py (NEW FILE):**

Implement `SecurityHeadersMiddleware` as a `BaseHTTPMiddleware` subclass exactly as in the research Pattern 4:
- Add headers to every response: `X-Frame-Options: DENY`, `X-Content-Type-Options: nosniff`, `X-XSS-Protection: 1; mode=block`, `Referrer-Policy: strict-origin-when-cross-origin`, `Permissions-Policy: camera=(), microphone=(), geolocation=()`.
- Only add `Strict-Transport-Security: max-age=31536000; includeSubDomains` when `not settings.DEBUG` (prevents HSTS on localhost).
- Add `Content-Security-Policy: default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' data: blob:; font-src 'self'; connect-src 'self'; frame-ancestors 'none'`.
- Import settings from `app.config`.

**5. Create backend/app/middleware/logging.py (NEW FILE):**

Implement `RequestLoggingMiddleware` as a `BaseHTTPMiddleware` subclass exactly as in the research Pattern 5:
- At start of dispatch: call `structlog.contextvars.clear_contextvars()` (prevents context leaking between requests).
- Bind context vars: `request_id` from `correlation_id.get()` (from asgi-correlation-id), `method`, `path`, `client_ip`.
- Time the request with `time.perf_counter()`.
- On success: log `"request_completed"` at INFO with `status_code` and `duration_ms`.
- On exception: log `"request_failed"` at ERROR with `duration_ms`, `error`, and `exc_info=True`. Re-raise the exception.
