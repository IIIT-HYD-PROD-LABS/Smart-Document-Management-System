3. Create `backend/app/services/llm/schemas.py` with Pydantic v2 models:
   ```python
   class ExtractedField(BaseModel):
       """A single extracted field with confidence score."""
       value: str | float | list[str] | None = None
       confidence: float = Field(ge=0.0, le=1.0, description="Confidence 0-1")
       source_hint: str | None = Field(None, description="Where in the text this was found")

   class DocumentExtraction(BaseModel):
       """Complete structured extraction result from a document."""
       dates: list[ExtractedField] = Field(default_factory=list)
       amounts: list[ExtractedField] = Field(default_factory=list)
       parties: list[ExtractedField] = Field(default_factory=list, description="People or organizations mentioned")
       key_terms: list[ExtractedField] = Field(default_factory=list, description="Key clauses, references, identifiers")
       summary: str = Field("", description="One-paragraph summary of the document")
       overall_confidence: float = Field(0.0, ge=0.0, le=1.0)
   ```
   These models serve double duty: they are Pydantic models for instructor's `response_model` AND for JSON serialization to the database.

4. Create `backend/app/services/llm/provider.py`:

   ```python
   def get_llm_client(provider: str, api_key: str, model_name: str | None = None) -> tuple[Any, str]:
       """Return (instructor_client, model_name) for the given provider.

       Supports: "openai", "anthropic".
       Raises ValueError for unsupported providers or missing API keys.
       """
   ```
   Implementation:
   - For "gemini": `import google.generativeai as genai; genai.configure(api_key=api_key); client = instructor.from_gemini(genai.GenerativeModel(model_name or "gemini-2.0-flash"))`, default model = `model_name or "gemini-2.0-flash"`.
   - For "openai": `client = instructor.from_openai(OpenAI(api_key=api_key))`, default model = `model_name or "gpt-4o-mini"`.
   - For "anthropic": `client = instructor.from_anthropic(Anthropic(api_key=api_key))`, default model = `model_name or "claude-haiku-4-5-20251001"`.
   - Raise `ValueError(f"Unsupported LLM provider: {provider}")` for anything else.
   - Do NOT import these at module level. Use lazy imports inside the function to avoid import errors when SDKs aren't installed.
