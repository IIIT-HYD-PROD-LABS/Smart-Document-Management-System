  - `extract_amounts(text)` -> list of {amount, currency} dicts for INR/USD
  - `extract_vendor(text)` -> string or None, using keyword search then first-line heuristic

  Key design decisions:
  - `dayfirst=True` for Indian DD/MM/YYYY format
  - Year validation (2000-2030) prevents false positives
  - Amount validation (0.01-10M) prevents reference numbers
  - Deduplication using dict.fromkeys
  - Uses structlog for logging
