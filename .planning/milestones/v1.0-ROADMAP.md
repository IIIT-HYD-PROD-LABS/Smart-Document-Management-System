## Overview

AI-powered Smart Document Management System that automatically classifies uploaded documents using ML, extracts text via OCR, performs LLM-powered metadata extraction, and provides full-text search with fuzzy matching. Features role-based access control, OAuth SSO, document sharing, version control, analytics dashboard, and comprehensive audit logging.

## Milestone Summary

**Key Decisions:**
- Opaque refresh tokens (not JWT) for server-side revocability
- LinearSVC with CalibratedClassifierCV for probability-calibrated classification
- PostgreSQL TSVECTOR + trigram for search (no Elasticsearch dependency)
- LLM provider chain with automatic fallback (ollama → gemini → local)
- JSONB for extracted metadata (flexible schema)
- Celery with prefork pool and memory limits for document processing

**Key Accomplishments:**
1. Secure auth with token rotation, reuse detection, and OAuth SSO (Google + Microsoft)
2. Full OCR/PDF/DOCX pipeline with async Celery processing and progress tracking
3. ML classification at 85% accuracy on 7 real Kaggle datasets
4. Full-text search with fuzzy matching, filters, and sub-2s response times
5. LLM-powered extraction with 5-provider fallback chain
6. Role-based access control with document-level sharing permissions
7. Analytics dashboard, version control, and document preview
8. 180+ automated tests, CI/CD pipeline, and production deployment docs

**Tech Debt Carried Forward:**
- Refresh tokens in JS-accessible cookies (not httpOnly)
- passlib is unmaintained
- No frontend test framework
- Offset-based pagination (not cursor-based)
